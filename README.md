# Job Interativo

Plataforma de Recrutamento com:
- Área do candidato
- Área da empresa
- Painel de vagas
- Painel administrativo
- Conteúdos e contato

## Deploy no Render

1. Suba o backend e frontend em repositórios separados.
2. Crie serviços Web no Render.
3. Configure a variável DATABASE_URL.
4. No frontend, defina VITE_API_URL apontando para o backend.
